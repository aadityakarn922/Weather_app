# Weather_app
A simple weather app that shows real-time temperature, humidity, and wind speed using the OpenWeather API.

This is a simple and clean weather application built using HTML, CSS, and JavaScript.  
It allows users to search for any city and instantly see the current temperature, humidity, and wind speed.

The app uses the OpenWeather API to fetch real-time weather data and updates the UI dynamically based on the weather conditions.

---

## Features
- Search weather by city name
- Shows current temperature in Celsius
- Displays humidity and wind speed
- Changes weather icons based on conditions (Clear, Clouds, Rain, Snow, Fog, etc.)
- Handles invalid city names with an error message
- Search using button click or Enter key
- Clean and responsive card-based UI

---

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- OpenWeather API
